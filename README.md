📍 Puedes leer el contenido en:

👉 [https://benji-rodriguez.com/blog](https://benji-rodriguez.com/blog)

---

## 🧱 Sobre la tecnología

Este blog está construido sobre **[BlogItMD](https://github.com/Bbenrod/BlogItMD)**, un CMS minimalista que yo mismo desarrollé usando:

- Node.js
- Express
- EJS
- CSS
- express-session
- Redis (para sesiones en producción)

Gracias a su diseño modular y simple, BlogItMD me permite publicar entradas de manera rápida, clara y con control completo sobre el contenido y estilo.

---

## 🔄 Actualizaciones

Este repositorio mantiene una referencia al proyecto original como `upstream`, lo que me permite incorporar fácilmente mejoras al motor base:

```bash
git remote add upstream git@github.com:Bbenrod/BlogItMD.git
git pull upstream main
````
